# AI Trip Planner

AI Trip Planner is a sophisticated tool designed to assist users in planning their trips efficiently. It leverages advanced AI technologies to provide personalized travel recommendations, itinerary planning, and real-time updates.

## Features

- **Personalized Itinerary Planning**: Generate customized travel itineraries based on user preferences and constraints.
- **Real-Time Weather Updates**: Get up-to-date weather information for your travel destinations.
- **Currency Conversion**: Easily convert currencies using the latest exchange rates.
- **Place Search**: Discover popular places and attractions at your destination.
- **Expense Calculation**: Keep track of your travel expenses with ease.

## Installation

To install the AI Trip Planner, ensure you have Python 3.14 or higher installed. Then, clone the repository and install the dependencies:

```bash
git clone https://github.com/Harikrishnan46624/AI-TRIP-PLANNNER.git
cd AI-TRIP-PLANNNER
pip install -r requirements.txt
```

## Usage

Run the application using the following command:

```bash
python app.py
```
